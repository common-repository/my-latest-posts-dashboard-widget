This plugin displays the latest posts by the currently logged-in user in a dashboard widget.

== Description ==

If your blog has multiple users, this widget will display posts by the currently logged-in user in a dashboard widget.

== Installation ==

1. Upload `latest-posts-dashboard-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Adjust options by clicking 'Configure' in the top right-hand corner of the widget
